# Maintenance status

This project is published as an unmaintained personal project.

The author does not plan to provide releases, compatibility updates, security
patches, support, issue triage, or pull-request reviews. There is no response or
resolution-time commitment. The repository may be archived without notice.

You are welcome to:

- use the source at your own risk;
- report observations or suggest improvements, understanding that they may not
  receive a response;
- fork the project and maintain it independently;
- reuse the MIT-licensed Rust implementation with attribution through the
  included license.

This project is not affiliated with IIT Roorkee, Cisco, or the OpenConnect
project. IITR may change its gateway, authentication flow, routes, or access
policy at any time.
